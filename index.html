Quick Start — Docker

git clone https://github.com/Steven5233/vulnora.git
cd vulnora
cp backend/.env.example backend/.env
docker compose up --build -d

UI: http://localhost:8501   •   API: http://localhost:8000   •   Proxy: http://localhost:8090
