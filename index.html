Vulnora v2.1.0

Professional-grade, open-source hybrid vulnerability scanning and manual pentesting platform. Bridging automated DAST with global compliance frameworks.

Get Started Now

✨ What's New in v2.1.0

  • Advanced IDOR/BOLA Engine: Specialized logic for GraphQL manipulation and cross-session manipulation.
  • LogicFlawScanner: Enhanced with IDORForgeProScanner.
  • Dynamic Test Accounts: Automated registration and session switching for deep auth testing.
  • Better Discovery: Improved _smart_discovery for JSON object ID extraction.

Platform Capabilities

Automated Engine

Includes Nuclei for CVEs, OWASP ZAP for spidering, and Subfinder/Nmap for full-spectrum recon.

Manual Pentesting

Integrated ZAP Proxy & Repeater. Edit and resend requests, manage JWTs, and view live traffic history.

Compliance Ready

Automated mapping to NIST CSF, ISO 27001, GDPR, PCI DSS, and SOC 2.

Advanced Logic

Detects business logic flaws like price manipulation and workflow bypasses that traditional scanners miss.

Quick Start (Docker)

git clone https://github.com/Steven5233/vulnora.git
cd vulnora
cp backend/.env.example backend/.env
docker compose up --build -d

Access UI: http://localhost:8501 | API: http://localhost:8000

Advanced Logic & Auth Module

Vulnora v2.1.0 performs real, authenticated HTTP requests to uncover high-impact vulnerabilities:

Multi-Account Manipulation: Dynamically tests cross-user access and privilege escalation.

GraphQL Mutations: Deep testing of nested objects and unauthorized data modifications.
